# Bienvenue dans Harzourou

**Harzourou** est un langage de programmation moderne, expressif et multi-paradigme, conçu pour la data science, le calcul tensoriel et la programmation fonctionnelle. Développé à l'ENSAM Meknès par C. Aboulaye, il s'inspire des langages de flux de données avec une sémantique proche de Python, Kotlin et OCaml.

<Callout type="info">
  Harzourou est actuellement en développement actif. Cette documentation couvre la version **1.0.0**.
</Callout>

## Pourquoi Harzourou ?

- **Déclaration de flux** — les variables sont déclarées avec le mot-clé `flow`, reflétant la nature du flux de données.
- **Typage précis** — types numériques de `int8` à `float64`, booléens, chaînes.
- **Modules natifs riches** — `math`, `string`, `list`, `func`, `io`, `net`, `json`, `time`, `random`, et bien plus.
- **Programmation fonctionnelle** — `map`, `filter`, `fold`, closures, opérateur pipe `|>`, composition `>>`.
- **Gestion des erreurs** — `try / catch / finally / throw / rethrow`.
- **Pattern matching** — `match / case / default`.
- **Modules & imports** — système modulaire avec `import`, `from … as`.
- **Aiki** — gestionnaire de paquets officiel pour gérer dépendances, scripts et builds.

## Premier programme

```harzourou
// Bonjour le monde  —  hello.harz
fprintln("Bonjour, Harzourou !")

// Variables et calcul
flow x   : int    = 42
flow pi  : float  = 3.14159
flow msg : string = "résultat"

fprintln(msg ~ " : " ~ str(x))

// Pipeline fonctionnel
flow résultat = [1..=10]
    |> filter(fun(n){ return n % 2 == 0 })
    |> map(fun(n){ return n * n })
    |> fold(0, fun(acc, n){ return acc + n })

fprintln("Somme des carrés pairs : " ~ str(résultat))  // 220
```

Exécuter avec :

```bash
harz hello.harz
```
