## Publication du fichier

Pour diffuser l'installateur depuis ce site, placer le binaire dans :

```text
public/downloads/Harzourou-Setup-Windows-x64.exe
```

Le lien public sera automatiquement :

```text
/downloads/Harzourou-Setup-Windows-x64.exe
```

## Vérifier le checksum (Windows)

```powershell
Get-FileHash .\Harzourou-Setup-Windows-x64.exe -Algorithm SHA256
```

## Build depuis les sources

Harzourou peut être compilé sur Windows, Linux et macOS avec CMake :

```bash
git clone https://github.com/harzourou/harzourou
cd harzourou
cmake -S . -B build -DCMAKE_BUILD_TYPE=Release
cmake --build build --config Release
```

Le binaire produit se nomme `harz` (ou `harz.exe` sur Windows).
