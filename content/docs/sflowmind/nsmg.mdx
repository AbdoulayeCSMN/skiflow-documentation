# Module `nsmg` — NeuroSymbolic Memory Graph

Le **NSMG** est un graphe de connaissances orienté pondéré avec :
- propagation d'activation par BFS (spreading activation),
- apprentissage différentiable par rétropropagation (SGD),
- recherche approximée de plus proches voisins via **HNSW**,
- embeddings vectoriels alignés 32 octets (compatibles AVX2),
- persistance binaire `.aidb`.

## Import

```skiflow
import nsmg
```

---

## Cycle de vie

### `nsmg.new(embed_dim)` -> graph

Crée un graphe vide. `embed_dim` est la dimension des vecteurs d'embedding (doit être un multiple de 8 pour le SIMD).

```skiflow
flow g = nsmg.new(128)
```

---

## Construction du graphe

### `nsmg.add(g, label)` -> node_id

Ajoute un nœud concept et retourne son identifiant entier stable.

```skiflow
flow chien = nsmg.add(g, "chien")
flow chat  = nsmg.add(g, "chat")
flow animal= nsmg.add(g, "animal")
```

### `nsmg.link(g, from, to, weight, edge_type)` -> null

Ajoute un arc dirigé pondéré entre deux nœuds.

| `edge_type` | Valeur | Signification |
|-------------|--------|---------------|
| IS_A | 0 | Relation taxonomique (sous-type) |
| PART_OF | 1 | Relation méréologique (partie de) |
| CAUSES | 2 | Dépendance causale |
| RELATED_TO | 3 | Association générique |
| CUSTOM | 4 | Catégorie applicative |

```skiflow
nsmg.link(g, chien,  animal, 0.9, 0)   // chien IS_A animal
nsmg.link(g, chat,   animal, 0.9, 0)   // chat  IS_A animal
nsmg.link(g, animal, chien,  0.5, 3)   // lien générique
```

### `nsmg.node_count(g)` -> int

```skiflow
fprintln nsmg.node_count(g)   // nombre de nœuds
```

### `nsmg.label(g, node_id)` -> string

```skiflow
fprintln nsmg.label(g, chien)   // "chien"
```

### `nsmg.find(g, label)` -> int

Recherche O(1) par label via hash map interne. Retourne `-1` si absent.

```skiflow
flow id = nsmg.find(g, "chat")
```

### `nsmg.clock(g)` -> int

Retourne l'horloge logique courante (incrémentée par `decay`).

---

## Activation & propagation

### `nsmg.activate(g, node_id, strength)` -> null

Initialise l'activation d'un nœud.

```skiflow
nsmg.activate(g, chien, 1.0)
```

### `nsmg.spread(g, node_id)` -> null

Propagation BFS depuis `node_id`. Chaque voisin reçoit :
`delta = activation_source × poids_arc`, tant que `delta` dépasse le seuil interne.

```skiflow
nsmg.spread(g, chien)
fprintln nsmg.activation(g, animal)   // ≈ 0.9
```

### `nsmg.activation(g, node_id)` -> float

Lit le niveau d'activation courant d'un nœud.

### `nsmg.decay(g, factor)` -> null

Multiplie toutes les activations par `factor` et avance l'horloge d'un tick.

```skiflow
nsmg.decay(g, 0.9)   // oubli léger
```

### `nsmg.reset(g)` -> null

Remet toutes les activations à zéro (topologie préservée).

---

## Apprentissage Hebbien

### `nsmg.reinforce(g, from, to, delta)` -> null

Renforce le poids de l'arc `from -> to` de `delta` (clampé à `[0, 1]`).

### `nsmg.weaken(g, from, to, delta)` -> null

Affaiblit le poids de l'arc `from -> to` de `delta`.

### `nsmg.forget(g, max_age)` -> null

Divise par deux l'activation des nœuds dont le dernier usage date de plus de `max_age` ticks.

---

## Embeddings

### `nsmg.emb_get(g, node_id)` -> list\[float\]

Retourne l'embedding du nœud sous forme de liste de floats.

```skiflow
flow emb = nsmg.emb_get(g, chien)   // [f1, f2, ..., f128]
```

### `nsmg.emb_set(g, node_id, list)` -> null

Remplace l'embedding d'un nœud.

### `nsmg.nn(g, query_list, k)` -> list\[int\]

Recherche bruteforce des `k` plus proches voisins par distance euclidienne.

```skiflow
flow voisins = nsmg.nn(g, nsmg.emb_get(g, chien), 3)
```

---

## Apprentissage différentiable (SGD)

<Callout type="info">
  Appeler `nsmg.diff_init` avant tout `forward` / `backward` / `step`.
</Callout>

### `nsmg.diff_init(g, lr)` -> null

Initialise le moteur de rétropropagation avec le taux d'apprentissage `lr`.

```skiflow
nsmg.diff_init(g, 0.05)
```

### `nsmg.forward(g, seed_node, steps)` -> list\[float\]

Passe forward BFS depuis `seed_node` sur `steps` étapes.
Retourne un vecteur d'activations de taille `node_count`.

```skiflow
flow actvs = nsmg.forward(g, chien, 3)
fprintln "activation[animal] = " ~ to_string(actvs[animal])
```

### `nsmg.backward(g, grad_list)` -> null

Rétropropage le gradient (liste de `node_count` floats).

```skiflow
// Gradient MSE : pousser animal vers 1.0
fun make_grad(i) {
    if i == animal { return -2.0 * (1.0 - actvs[animal]) }
    else           { return 0.0 }
}
flow grad = func.range(0, nsmg.node_count(g) - 1)
    |> map(make_grad)

nsmg.backward(g, grad)
```

### `nsmg.step(g)` -> null

Applique la mise à jour SGD (poids ← poids - lr × gradient).

### `nsmg.zero_grad(g)` -> null

Remet les gradients à zéro avant la prochaine passe.

---

## Index HNSW (ANN accéléré)

L'index **HNSW** (Hierarchical Navigable Small World) remplace la recherche bruteforce O(n·d) par une recherche approximée O(log n).

### `nsmg.hnsw_build(g)` -> hnsw_index

Construit l'index sur les embeddings courants. À rappeler après ajout de nœuds.

```skiflow
flow idx = nsmg.hnsw_build(g)
```

### `nsmg.hnsw_search(idx, g, query_list, k)` -> list\[int\]

Recherche les `k` voisins les plus proches dans l'index.

```skiflow
flow query   = nsmg.emb_get(g, infection)
flow voisins = nsmg.hnsw_search(idx, g, query, 3)

fprintln nsmg.label(g, voisins[0])
fprintln nsmg.label(g, voisins[1])
fprintln nsmg.label(g, voisins[2])
```

### `nsmg.hnsw_insert(idx, g, node_id)` -> null

Insère un nœud dans un index déjà construit (mise à jour incrémentale).

---

## Persistance

```skiflow
nsmg.save(g, "knowledge.aidb")   // -> int (0 = succès)
flow g2 = nsmg.load("knowledge.aidb")
```

## Exemple complet — graphe médical

```skiflow
import nsmg
import func

flow g = nsmg.new(128)

// Concepts
flow fievre    = nsmg.add(g, "fièvre")
flow infection = nsmg.add(g, "infection")
flow virus     = nsmg.add(g, "virus")
flow bacterie  = nsmg.add(g, "bactérie")
flow antibio   = nsmg.add(g, "antibiotique")
flow vaccin    = nsmg.add(g, "vaccin")
flow paraceta  = nsmg.add(g, "paracétamol")

// Relations
nsmg.link(g, virus,     infection, 0.9, 2)
nsmg.link(g, bacterie,  infection, 0.8, 2)
nsmg.link(g, infection, fievre,    0.85, 2)
nsmg.link(g, fievre,    paraceta,  0.7, 3)
nsmg.link(g, virus,     vaccin,    0.9, 3)
nsmg.link(g, bacterie,  antibio,   0.95, 3)

// Propagation
nsmg.activate(g, fievre, 1.0)
nsmg.spread(g, fievre)

fprintln "activation(infection) = " ~ to_string(nsmg.activation(g, infection))
fprintln "activation(paracéta)  = " ~ to_string(nsmg.activation(g, paraceta))

// Apprentissage : pousser vaccin vers 1.0 (1 étape SGD)
nsmg.diff_init(g, 0.05)
flow actvs = nsmg.forward(g, virus, 3)
fprintln "vaccin avant = " ~ to_string(actvs[vaccin])

flow grad = func.range(0, nsmg.node_count(g) - 1)
    |> map(fun(i){ return i == vaccin ? -2.0 * (1.0 - actvs[vaccin]) : 0.0 })

nsmg.backward(g, grad)
nsmg.step(g)
nsmg.zero_grad(g)

flow actvs2 = nsmg.forward(g, virus, 3)
fprintln "vaccin après = " ~ to_string(actvs2[vaccin])

// HNSW
flow idx     = nsmg.hnsw_build(g)
flow query   = nsmg.emb_get(g, infection)
flow voisins = nsmg.hnsw_search(idx, g, query, 3)
fprintln "Voisins HNSW de 'infection' :"
fprintln "  -> " ~ nsmg.label(g, voisins[0])
fprintln "  -> " ~ nsmg.label(g, voisins[1])
fprintln "  -> " ~ nsmg.label(g, voisins[2])

nsmg.save(g, "medical.aidb")
```
