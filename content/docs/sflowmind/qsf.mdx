# Module `qsf` — Quantum Semantic Field

Le **QSF** modélise des états sémantiques comme des vecteurs d'amplitudes **complexes** dans un espace de Hilbert de dimension `dim`. Il implémente :
- **superposition** de concepts (somme d'états),
- **interférence** (constructive / destructive),
- **collapse** déterministe (argmax) ou stochastique (échantillonnage Born),
- **évolution unitaire** (rotation de phase),
- stockage **dense** ou **sparse** automatique (< 10 % de dimensions non nulles),
- **intrication** (corrélation non-locale entre paires de waves),
- **FFT sémantique** et spectre de puissance.

<Callout type="info">
  Les amplitudes sont des nombres complexes ℂ. La probabilité d'un concept `d` est `|ψ[d]|²` (règle de Born).
</Callout>

## Import

```skiflow
import qsf
```

---

## Cycle de vie

### `qsf.new(dim)` -> field

Crée un champ sémantique de dimension `dim`.

```skiflow
flow f = qsf.new(256)
```

---

## Gestion des waves

### `qsf.wave(f, label)` -> wave_id

Crée une wave sémantique (vecteur d'amplitudes initialisé uniformément).

```skiflow
flow w_chien  = qsf.wave(f, "chien")
flow w_chat   = qsf.wave(f, "chat")
flow w_animal = qsf.wave(f, "animal")
```

### `qsf.wave_count(f)` -> int

Nombre de waves dans le champ.

### `qsf.label(f, wave_id)` -> string

Retourne le label associé à une wave.

---

## Opérations quantiques

### `qsf.superpose(f, wave_a, wave_b, alpha, beta)` -> wave_id

Crée une wave résultant de la superposition :
`ψ_new = α·ψ_a + β·ψ_b` (normalisée automatiquement).

```skiflow
flow w_super = qsf.superpose(f, w_chien, w_chat, 0.7, 0.3)
```

### `qsf.interfere(f, wave_a, wave_b)` -> wave_id

Interférence quantique directe entre deux waves.
Les phases alignées -> **interférence constructive** (amplification).
Les phases opposées -> **interférence destructive** (annulation).

```skiflow
flow w_int = qsf.interfere(f, w_animal, w_super)
```

### `qsf.evolve(f, wave_id, dt)` -> null

Applique une rotation de phase `e^{iωt}` à chaque dimension (évolution unitaire, préserve la norme).

```skiflow
qsf.evolve(f, w_chien, 0.1)
```

### `qsf.normalize(f, wave_id)` -> null

Re-normalise la wave pour que `Σ|ψ[d]|² = 1`.

---

## Mesure & observation

### `qsf.observe(f, wave_id)` -> int

Collapse la fonction d'onde selon la règle de Born :
- Mode **argmax** (déterministe) : retourne le concept `d` tel que `|ψ[d]|²` est maximal.
- Mode **sample** (probabiliste) : échantillonne selon la distribution `P(d) = |ψ[d]|²`.

```skiflow
flow concept = qsf.observe(f, w_super)
fprintln "concept observé = " ~ to_string(concept)
```

---

## Similarité & compression

### `qsf.similarity(f, wave_a, wave_b)` -> float

Similarité quantique via le produit scalaire complexe (Born rule) :
`S = |⟨ψ_a|ψ_b⟩|² ∈ [0, 1]`.

```skiflow
flow sim = qsf.similarity(f, w_chien, w_chat)
fprintln "Similitude chien/chat = " ~ to_string(sim)
```

### `qsf.compress(f, wave_id, threshold)` -> null

Passe la wave en représentation **sparse** : met à zéro toutes les dimensions `d` telles que `|ψ[d]|² < threshold`.

```skiflow
qsf.compress(f, w_animal, 0.01)
```

---

## Analyse spectrale

### `qsf.convolve(f, wave_a, wave_b)` -> wave_id

Convolution de deux waves dans l'espace de Fourier (produit de Hadamard des FFT).

### `qsf.power_spectrum(f, wave_id)` -> list\[float\]

Retourne le spectre de puissance `|FFT(ψ)[k]|²` pour chaque mode `k`.

```skiflow
flow spectrum = qsf.power_spectrum(f, w_chien)
fprintln "Mode dominant : " ~ to_string(func.maximum(spectrum))
```

---

## Intrication

L'**intrication** corrèle deux waves de façon non-locale : observer l'une modifie instantanément l'état de l'autre.

### `qsf.ent_new(f, wave_a, wave_b)` -> pair_id

Crée une paire intriquée standard (état de Bell).

### `qsf.ent_pair(f, wave_a, wave_b)` -> pair_id

Crée une intrication par produit tensoriel.

### `qsf.ent_antonym(f, wave_a, wave_b)` -> pair_id

Intrication antonyme — état anti-corrélé (observer `a = d` implique `b ≠ d`).

### `qsf.ent_custom(f, wave_a, wave_b, phase)` -> pair_id

Intrication avec déphasage personnalisé `phase` ∈ \[0, 2π\].

### `qsf.ent_observe(f, pair_id, which)` -> int

Observe l'un des membres (`which` = 0 ou 1) et effondre les deux.

### `qsf.ent_collapse(f, pair_id)` -> null

Force l'effondrement de la paire (libère la corrélation).

qsf.save(f, "semantic.aidb")

## Exemple complet — sémantique quantique

```skiflow
import qsf

flow f = qsf.new(128)

// Créer les waves
flow w_roi    = qsf.wave(f, "roi")
flow w_reine  = qsf.wave(f, "reine")
flow w_homme  = qsf.wave(f, "homme")
flow w_femme  = qsf.wave(f, "femme")

// Analogie vectorielle quantique : roi - homme + femme ≈ reine
flow w_analogy = qsf.superpose(f, w_roi, w_homme, 1.0, -1.0)
flow w_result  = qsf.superpose(f, w_analogy, w_femme, 1.0, 1.0)

// Similarité avec reine
flow sim = qsf.similarity(f, w_result, w_reine)
fprintln "Analogie roi-homme+femme ≈ reine : " ~ to_string(sim)

// Observation
flow observed = qsf.observe(f, w_result)
fprintln "Concept observé : " ~ qsf.label(f, observed)

// Intrication : homme ↔ femme (antonymes)
flow pair = qsf.ent_antonym(f, w_homme, w_femme)
flow obs_a = qsf.ent_observe(f, pair, 0)
fprintln "Observation homme = " ~ to_string(obs_a)
// La wave femme est instantanément effondrée dans l'état opposé

// Spectre de puissance
flow spectrum = qsf.power_spectrum(f, w_roi)
fprintln "Modes : " ~ to_string(qsf.wave_count(f))

qsf.save(f, "semantics.aidb")
```
