# Module `json`

Sérialisation et désérialisation JSON.

## Import

```harzourou
import json
```

## Fonctions

```harzourou
json.parse(s)         // chaîne JSON → valeur Harzourou
json.stringify(val)   // valeur Harzourou → chaîne JSON
json.pretty(val)      // JSON indenté
```

## Exemple

```harzourou
import json

flow data = json.parse('{"nom": "Amine", "score": 99}')
fprintln(data["nom"])    // Amine

flow sortie = json.stringify([1, 2, {"clé": true}])
fprintln(sortie)         // [1,2,{"clé":true}]
```
