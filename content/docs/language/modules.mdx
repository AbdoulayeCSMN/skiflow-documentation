# Modules & Imports

Harzourou dispose d'un système modulaire complet pour organiser le code en unités réutilisables.

## Importer un module natif

```harzourou
import math
import string
import list
import func
import io
import json
import net
import time
import random
```

## Utiliser un module

```harzourou
import math

flow racine = math.sqrt(16.0)        // 4.0
flow angle  = math.sin(math.pi / 2)  // 1.0
```

## Import sélectif (`from … import`)

```harzourou
from math import {sqrt, pi, sin, cos}

flow r = sqrt(25.0)  // 5.0
```

## Alias (`as`)

```harzourou
import math as m

flow v = m.sqrt(9.0)   // 3.0
```

## Définir un module (fichier `.mharz`)

Un module est un fichier `.mharz` commençant par le mot-clé `module` :

```harzourou
// geometrie.mharz
module geometrie

fun aire_cercle(r : float) : float {
  import math
  return math.pi * r * r
}

fun perimetre(r : float) : float {
  import math
  return 2.0 * math.pi * r
}
```

## Importer son propre module

```harzourou
// main.harz
import geometrie

flow a = geometrie.aire_cercle(5.0)   // 78.539…
flow p = geometrie.perimetre(5.0)     // 31.415…
fprintln("Aire : " ~ str(a))
fprintln("Périmètre : " ~ str(p))
```

<Callout type="info">
  Harzourou cherche les modules dans l'ordre :
  1. Le dossier du fichier courant
  2. `HARZOUROU_PKG_PATH` (packages installés par Aiki)
  3. `HARZOUROU_PATH` (bibliothèque standard)
</Callout>

## Modules natifs disponibles

| Module | Description |
|--------|-------------|
| `math` | Fonctions mathématiques et constantes |
| `string` | Manipulation de chaînes |
| `list` | Opérations sur les listes persistantes |
| `func` | Utilitaires fonctionnels (zip, flatten, chunks…) |
| `io` | Entrées/sorties fichiers |
| `json` | Sérialisation/désérialisation JSON |
| `net` | Réseau HTTP/TCP |
| `time` | Date, heure, timestamp |
| `random` | Génération aléatoire |
| `path` | Chemins de fichiers |
| `kern` | Noyaux de convolution |
| `csv` | Lecture/écriture CSV |
| `flare` | Visualisation (graphiques HTML) |
| `fluxnet` | Réseaux de neurones |
| `arithmetic` | Arithmétique étendue (is_prime, pgcd, etc.) |
| `geometry` | Primitives géométriques 2D/3D |
| `datetime` | Calculs de date avancés |
| `dstats` | Statistiques descriptives |
| `assert` | Framework de tests unitaires |
| `collections` | Stack, Queue, Deque, Pair |

## Gestion avec Aiki

Les modules tiers sont gérés par **Aiki** :

```bash
# Ajouter une dépendance
aiki add math

# Le module est ensuite disponible dans main.harz
```

```harzourou
// main.harz — après aiki add math
import math
fprintln(math.sqrt(2.0))  // 1.4142…
```
