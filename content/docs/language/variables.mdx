# Variables & Types

En Harzourou, toutes les variables sont déclarées avec le mot-clé `flow`. L'opérateur d'assignation est `=` ou `<-` selon le contexte.

## Déclaration

```harzourou
flow nom : type = valeur     // déclaration typée
flow nom <- valeur           // inférence de type
flow nom                     // null implicite
```

### Exemples

```harzourou
flow age    : int    = 25
flow pi     : float  = 3.14159
flow actif  : bool   = true
flow prénom : string = "Amine"
flow vide                       // null implicite
```

## Types primitifs

| Type | Description | Exemple |
|------|-------------|---------|
| `int` | Entier 32-bit | `42` |
| `float` | Flottant 64-bit | `3.14` |
| `bool` | Booléen | `true`, `false` |
| `string` | Chaîne de caractères | `"bonjour"` |
| `char` | Caractère unique | `'a'` |
| `null` | Valeur nulle | `null` |

## Types numériques précis

Harzourou supporte des types numériques précis, essentiels pour la data science et le calcul haute performance :

| Entiers signés | Entiers non signés | Flottants |
|---|---|---|
| `int8`, `int16`, `int32`, `int64` | `uint8`, `uint16`, `uint32`, `uint64` | `float16`, `bfloat16`, `float32`, `float64` |

```harzourou
flow poids  : float32  = 0.5
flow pixels : uint8    = 255
flow grad   : bfloat16 = 1.0e-3
flow count  : int64    = 1_000_000_000
```

## Inférence de type

Le compilateur infère le type lorsqu'on n'en précise pas :

```harzourou
flow x = 10        // int
flow y = 3.14      // float
flow ok = true     // bool
flow s = "hello"   // string
```

## Listes

```harzourou
flow nombres : list = [1, 2, 3, 4, 5]
flow mixte   = [1, "deux", true, null]
flow vide    = []

// Accès par index (0-based, négatif depuis la fin)
flow premier  = nombres[0]   // 1
flow dernier  = nombres[-1]  // 5
```

## Ranges

```harzourou
flow r1 = 1..10     // [1, 2, …, 9]   exclusif
flow r2 = 1..=10    // [1, 2, …, 10]  inclusif
flow r3 = 0..=4     // [0, 1, 2, 3, 4]
```

## Chaîne de caractères

L'opérateur `~` concatène des chaînes :

```harzourou
flow prénom = "Amine"
flow message = "Bonjour, " ~ prénom ~ " !"
fprintln(message)   // Bonjour, Amine !

// Conversion en chaîne
flow n = 42
fprintln("Valeur : " ~ str(n))
```

## Mutation et `prev`

Par défaut les variables sont mutables. Le mot-clé `prev` accède à la valeur précédente dans un contexte de flux :

```harzourou
flow compteur = 0
compteur = compteur + 1

// prev dans une boucle
while compteur < 10 {
  flow delta = compteur - prev compteur
  compteur += 1
}
```

<Callout type="warning">
  `prev` est uniquement disponible dans les contextes de flux temporel (boucles, unités).
</Callout>

## Portée des variables

Les variables sont locales au bloc dans lequel elles sont déclarées :

```harzourou
flow x = 10

if x > 5 {
  flow y = x * 2   // y n'existe que dans ce bloc
  fprintln(y)       // 20
}

// fprintln(y)  ← erreur : y hors portée
```

## Constantes et immutabilité

En attendant un mot-clé `const` dédié, la convention est de nommer les constantes en MAJUSCULES et de ne jamais les réassigner :

```harzourou
flow PI      = 3.14159265358979
flow MAX_AGE = 150
```
